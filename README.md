# portal-main-archive

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库本身不针对任何特定域名或网站，仅作为静态 HTML 文件的集中存放与版本管理之用。

## 项目简介

本仓库旨在收集、整理并维护一批独立的 HTML 页面。这些页面可能来源于不同时期、不同用途，统一存放于此以便于查阅、回溯与分发。仓库内容以 HTML 文件为主，不包含动态后端逻辑。

## 目录说明

```
portal-main-archive/
├── pages/          # 存放主要 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、图片、字体等）
├── docs/           # 相关文档与说明（可选）
└── README.md       # 本文件
```

- `pages/`：包含多个独立的 HTML 文件，每个文件均为一个完整的页面。
- `assets/`：存放页面所需的共用资源，按需组织。
- `docs/`：可放置关于页面用途、归档背景等补充说明。

## 页面归档说明

- 仓库中的 HTML 页面均为独立文件，可直接在浏览器中打开。
- 页面内容可能来自不同来源，但均经过整理与归档，以保证基本可读性。
- 部分页面可能引用外部资源，建议在离线环境下使用时提前保存相关依赖。
- 归档不保证所有页面均保持最新状态，仅作为历史记录与资料留存。

## 维护说明

- 欢迎通过 Issues 提交关于页面缺失、链接失效或内容错误的问题。
- 如需新增页面，请将 HTML 文件放入 `pages/` 目录，并确保不引入外部不可控资源。
- 本仓库不接受包含恶意代码、广告或违反相关法律法规的内容。
- 维护者会不定期整理目录结构，清理无效或重复文件。
